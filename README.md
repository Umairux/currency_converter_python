# Currency Converter

A Python application that converts one currency into another using live exchange rates from an API.

## Features
- Convert between different currencies
- Uses real-time exchange rates
- Simple command-line interface

## Technologies Used
- Python
- Requests
- Frankfurter API

## Note
the list of available currencies depends on the Frankfurter API
PKR(pakistan rupees) is not currently not supported this API

##API Used
Frankfurter API
